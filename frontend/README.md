# 智能音箱客服 - Vue 3前端

这是一个基于Vue 3的智能音箱客服前端应用，具有小猫风格的主题设计。

## 特性

- 🎨 小猫风格设计（纯白+暗角）
- 🌙 支持亮色/暗色主题切换
- 💬 实时流式聊天体验
- 🤔 客服思考过程可视化
- 📱 响应式设计
- ⚡ Vue 3 + Vite构建

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 构建和部署

1. 运行 `npm run build` 构建应用
2. 构建的文件将输出到 `../static/dist/` 目录
3. FastAPI服务器会自动服务这些静态文件

## 主题说明

- **亮色主题**: 纯白背景配粉色小猫元素
- **暗色主题**: 深色背景配亮色小猫元素
- 主题会自动保存到本地存储

## 主要功能

- 智能客服对话
- 实时消息流式显示
- 客服思考状态指示
- 主题切换功能
- 自动文本框高度调整
